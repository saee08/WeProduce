# WeProduce

## What is WeProduce?
WeProduce is a productivity and accountability platform designed to track and accelerate the technical growth of a three-person cohort. 

It is a centralized tracking dashboard that automatically aggregates daily developer activities across multiple platforms, specifically **HackerRank** solutions, **LeetCode** problem-solving, and **GitHub** project commits. By translating these continuous efforts into a unified scoring system, the platform generates a weekly comparative leaderboard.

Ultimately, WeProduce turns the solitary act of building a tech profile into a collaborative, metrics-driven challenge that fosters healthy competition and ensures consistent daily coding habits.

## 💻 Tech Stack
- **Frontend:** React Native
- **Backend:** Next.js
- **UI Components:** ShadCN
- **Database:** Supabase
- **Authentication:** Google OAuth
- **Hosting:** Vercel

## 🔌 Integrations
- HackerRank
- LeetCode
- GitHub

## ✨ Must-Have Features
- 🌓 **Dark/Light Mode:** Seamless support for themed environments.
- 👤 **Individual Profiles:** Dedicated views for each of the three cohort members.
- 📊 **Dashboard:** A central hub displaying and comparing weekly scores.
- 🏆 **Scoring Metrics:** Unified points system translating cross-platform efforts.
- 💬 **Discussions:** Post and thread support for sharing insights and communicating.
- 📜 **Commit History UI:** Clear visual representation of GitHub activity.
- 🔥 **Individual Streak Tracker:** Tool to monitor and maintain daily coding habits.
